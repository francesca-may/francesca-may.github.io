Work

Here are some cool projects I've had the privilege to work on. Clicking on a project will open it up in an alternative tab.

Build2020

Project: Bespoke WordPress Site

After being provided a design, I drafted this site in HTML & CSS using Bootstrap. I then adapted the draft into a compatible WordPress template. This site also uses WordPress plugins like advanced custom fields for client convenience, making it simple and fast for them to maintain their own sections. I also learned how to create custom posts, to make it easier for the client to post both blog posts and patient stories separately.

Mortimer Childe

Project: Bespoke WordPress Site

This site was first drafted in HTML & CSS using Bootstrap, based on a design given to me, and was then converted into WordPress templates. This site also uses WordPress plugins such as gravity forms (to make user - client communication easy) and advanced custom fields (for convenient client editable sections).

Energy Connection Services

Project: Mobile Friendly Services Grid

Showing eight services on a mobile version of the site felt excessive to the client and therefore it was my job to make this easier on the eye and more practical. I used Bootstrap columns to create a more responsive and accessible version of the grid; making sure that four key services were visible.

Rob Graham Portfolio

Project: Bespoke GitHub Pages Site

I both designed and developed this project. The main brief for this site was to keep it 'simple' and 'clean', yet take the programmer's personality and quirks to make it personal to them. It was for this reason that I designed the colour palette to be kept classic, yet with a pop of a feature colour (a colour to match the programmer's logo & overall brand). I also added a slight fade to the page transition to further the idea that the site is clean, but still has a special touch.
