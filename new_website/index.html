About

Hi there! I'm Francesca, a front-end developer from the UK. I love playing with code and creating simple solutions for effective web pages. When I'm not being creative with code, I really enjoy creating digital art and playing video games; as well as a bit of yoga to relax the mind!

I like to develop with HTML5 and CSS3 (Bootstrap also comes along handy here!), adding some pizzaz with JavaScript and it's associated libraries; such as jQuery (I have dabbled with React and Angular!). I am very familiar with creating custom Wordpress websites, knowing how to use PHP and MySQL to compliment development. I'm not shy when it comes to learning - I have experience with C# and ASP.NET too! If there is something you need me to know, I'm happy to get on board.
